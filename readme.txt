== Short Description ==
A plugin to check the presence of keywords in Title, Meta Description, H1, H2, H3, and internal links on each post/page.

== Description ==

This plugin checks if three keywords are present in the Title, Meta Description, H1, H2, H3 tags, and internal links of each post or page. It provides a "Check Now" button to review these keywords for SEO purposes.

== Installation ==

1. Upload the `keyword-checker` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the post or page editor and enter your keywords in the "Keyword Checker" meta box.
4. Click the "Check Now" button to see if the keywords are present in Title, Meta Description, H1, H2, H3, and internal links.

== Frequently Asked Questions ==

= What does this plugin do? =

This plugin helps you check if your keywords are included in various parts of your posts/pages for better SEO optimization.

= How do I use the "Check Now" button? =

After entering your keywords, simply click the "Check Now" button to see the results. The plugin will analyze your post/page and provide feedback.
